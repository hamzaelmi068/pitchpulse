# Product Planning Prompt

Use this prompt for roadmap, feature scoping, or product-definition work.

## Goal

Refine PitchPulse feature scope, milestone planning, and delivery sequencing with attention to the June 2026 World Cup launch timeline.

## Instructions

- Read `../project.md`, `../memory.md`, and `../context/roadmap.md` first.
- Keep plans grounded in the current repo truth: active frontend exists; backend and AI architecture remain in discovery.
- Distinguish clearly between alpha essentials, beta enhancements, and launch hardening work.
- Call out dependencies that could block delivery if provider or architecture choices slip.

## Expected output

- Proposed scope by milestone
- Key dependencies and risks
- Concrete next decisions for the team
